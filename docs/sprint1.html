4 september 2024

Vandaag werkte ik met mijn team aan een breakdownschets van de website van de HvA. Een breakdownschets geeft een overzicht van de benodigde HTML-elementen, waardoor je beter voorbereid aan het bouwen van een website begint. Het helpt je te onderzoeken welke technieken nodig zijn om een ontwerp te realiseren. Bij een breakdownschets moet je ervoor zorgen dat je de juiste HTML-elementen gebruikt en blokken tekent die overeenkomen met de structuur en styling (CSS-properties). Handige bronnen om HTML-elementen en CSS properties te vinden: MDN HTML Elements. en MDN CSS Properties. Zie voor meer informatie Github.

Vandaag hebben we gespeeld met verschillende manieren in Javascript. We hebben bijvoorbeeld geleerd dat je met document.querySelector verschillende selectors in HTML kan selecteren. Daarna kan je er andere dingen mee laten doen zoals het veranderen van de heading. Ook heb ik geleerd wat een DOM (Document Object Model). Een DOM is een programmainterface die wordt gebruikt voor webpagina's. Het beschrijft de structuur van een HTML document en maakt het mogelijk om de inhoud en stijl van een website te veranderen door midden van Javascript. Zie voor meer informatie Github.
